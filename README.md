# EggReportGen
🥚 A clean, professional, Excel-like multi-agent egg sales report dashboard. 

## Features
*   **Multi-Agent Support**: Separate data tracking per agent using clean tabs.
*   **Auto-Save**: Automatic data persistence to your browser's `localStorage`.
*   **Calendar Date Picker**: Easily select dates with a native calendar popup.
*   **Dropdown Product Selection**: Quick select egg grades (সাদা ডিম এ গ্রেড, লাল ডিম এ গ্রেড, সাদা ডিম বি গ্রেড, লাল ডিম বি গ্রেড, হাসের ডিম).
*   **Grand Summary Dashboard**: Premium flat KPI cards (Total Qty, Price, Deposit, Dues) and comparison tables.
*   **Zero-Free Look**: Input fields hide zero values to maintain a clean spreadsheet style.
*   **FontAwesome Icons**: Premium UI elements with zero emojis.
